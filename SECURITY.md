# Security Policy

## Supported Versions

All versions under 0.7.0 are unsupported. Update ASAP.

Any `-e` versions are also unsupported. While they do recieve security updates for mainstream releases, they will not recieve `-e` release-specific updates. Consider using a mainstream release version (version without the `-e` suffix)

Any mainstream versions are supported. 

| Version | Supported          |
| ------- | ------------------ |
| 0.7.0 >=   | :white_check_mark: |
| x.x.x-e   | :x:                |
| < 0.7.0   | :x:                |

## Reporting a Vulnerability

Report vulnerabilities by shooting a email to @sohiearth, or opening an issue on this repo with the `security` tag. Duplicates will be closed.
